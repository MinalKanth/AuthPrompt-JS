# 📖 Overview

AuthPrompt JS is a simple JavaScript authentication demonstration that validates user credentials entered through browser prompts.

The project introduces core JavaScript concepts including:

* User input collection
* Conditional statements
* Authentication logic
* Browser alerts
* Console logging

This project is designed for educational purposes and helps developers understand the fundamentals of login validation.

# 🔍 How It Works

1. User enters an email address.
2. User enters a password.
3. JavaScript compares the entered values with predefined credentials.
4. Authentication result is displayed.

Example:

```text
Email: abc@gmail.com
Password: 12345
```

Result:

```text
Authentication successful. Welcome!
```

# ⚠ Educational Notice

This project performs validation entirely on the client side and should not be used in production environments.

Real-world authentication systems should:

* Store passwords securely
* Use encrypted connections
* Validate credentials on the server
* Implement session management
